  3. What are special entity characters?
    Special Entity characters are number codes used to display special characters like © and ¥.
  4. What are phrase tags?
    Phrase tags are tags that can change how text is written visually or to egive structure to that text.
  5. What is the difference between OL and UL Tags?
    The OL tag is an ordered list, where the items in the list are specifically ordered by number, letter, or roman numeral. The UL tag is an unordered list where the order of items in the list doesnt matter, and is not numbered.
  6. What is a DL?
    The DL tag is a description list. The list contains dt tags for the title, and dd tags for the specific description under each title.
