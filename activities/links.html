  • What is the root? What is the relationship between folder and file names and the path?
    The root is the topmost or starting directory of all the files. The path is the sequence in the directory between the folders where you can locate a specific file. It is separated by a slash
  • What is a default document?
    The default document is the specific file that a web server sends to your browser when a URL doesnt specify a file name.
  • Explain the difference between an absolute link and a relative link. Why would you use relative links?
    An absolute link provides the complete web address including the protocol and domain name. A relative link is only a partial path that directs you to a page relative to your current page. You would use relative links because it brings you to another page in your own directory instead of linking to a different page.

  • List the 3 rules with relative links.
