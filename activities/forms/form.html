1. The form tag has an "action" attribute. What does it do?
The action tag shows where the data from the form will be sent when the user submits the form.

2. The form tag has a "method" attribute. What does it do?
The method tag shows which method to send data to the link in the action tag. This can either be POST or GET.

3. What is the difference between a "POST" and a "GET"?
POST is used to send data that changes something in the database, while GET is used to search/filter/sort and does not change the database.
