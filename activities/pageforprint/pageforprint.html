1. Why is the viewport meta tag important?
The viewport meta tag is important for web designs that scale together with the users device and makes it look good on phones.

2. How can you adjust the CSS for printing?
By using an @media tag in the css file. You can also do a media query in the html head.

3. When would you want to adjust the CSS for printing?
Only when you create a webpage that people want to print like coupons or tickets.
