1. What is normal flow in HTML?
Normal flow is when each block tag takes the entire width.

2. What does float do? Why do you need to clear the float?
Float makes the elements below it to share the same line if theres space. You need to clear the float to prevent the next element to wrap around the same float.

3. What do absolute, fixed, and relative do?
Relative position changes where an element goes relative to its original position. Absolute position removes the element from normal flow and other elements now occupy its space. You get to control exactly where you want it to go. Fixed position is like the absolute position where its not part of normal flow, but the element stays in the same place even when you scroll down.
