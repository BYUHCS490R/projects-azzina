1. List out the different ways to select an item.
You can select items via tag name, descendants or children rule, their ID #, and their class.

2. What is the difference between an ID and a Class?
An ID is more specific than a class. An ID cannot be reused and is only on one specific item. A class can be used more than once on different items. The class uses a period, and the ID uses a hashtag. You can also have multiple classes on an item.

3. What is the difference between a SPAN tag and a DIV tag?
A DIV is an inline-block spaced item, and a SPAN is an inline spaced item. A DIV can group paragraphs together, while a SPAN is usually for words in a sentence.

4. What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?
A div tag functions just like the other tags except the header, main, and footer tags are semantic. This means they clearly explain what they do. A div is just like an imaginary box.

5. What are the different ways to select a color?
You can use the color name, RGB, Hex Code, and Short Hex Code.
